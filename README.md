# DocuSense
AI Powered Semantic PDF Classification <br>
<p>DocuSense is a desktop application that automatically scans folders, understands the content of PDF documents, and intelligently organizes them into meaningful categories. Instead of manually sorting hundreds of files, users can allow DocuSense to analyze document content and automatically structure their study materials or document repositories.
The system uses natural language processing and semantic embeddings to classify documents, detect duplicates, and enable intelligent search across files.<p><br>

<h1><b>Problem</b></h1><br>
<p>Students, researchers, and professionals often accumulate large numbers of PDF documents such as lecture notes, research papers, and reports. Traditional file systems rely only on file names and manual folder organization, which makes finding relevant documents time-consuming and inefficient.
DocuSense solves this problem by analyzing document content and automatically organizing files based on their topics.</p><br>

<h1><b>Features</b></h1>
<h2><b>Core Features</b></h2>
<ul>
<li>Scan folders for PDF documents</li>
<li>Extract text from PDF files</li>
<li>Automatically classify documents by topic</li>
<li>Organize documents into categorized folders</li>
<li>Bulk processing of multiple PDFs</li></ul>
<br>
<h2><b>Advanced Features</b></h2>
<ul>
<li>Semantic document classification
<li>Duplicate and similar document detection</li>
<li>Natural language document search</li>
<li>Real-time folder monitoring</li>
<li>Document similarity scoring</li>
<li>User-friendly desktop interfac</li>
</ul>
<br>
<h1><b>System Workflow</b></h1><br>
<pre>User selects a folder
        ↓
Application scans for PDF files
        ↓
Text extracted from PDFs
        ↓
Content processed using AI embeddings
        ↓
Documents classified by topic
        ↓
Folders created automatically
        ↓
Files organized into categories</pre>
